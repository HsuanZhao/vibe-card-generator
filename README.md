# 🧩 Vibe 卡片生成器

> 每日一个 Vibe 工具 · 小红书 3:4 分享卡片

一键生成具有科技感、适合社交媒体传播的工具介绍卡片。输入网址即可自动抓取网站信息，并支持高度自定义，导出高清 PNG 图片。

## ✨ 功能特色

- 🔗 **智能抓取**：粘贴网站链接，自动读取标题、描述、域名及 Logo（通过 Clearbit 服务）。
- 🎨 **自由编辑**：所有文本内容（标题、tagline、详细描述、标签、期数、日期）均可手动调整。
- 🖌️ **配色主题**：内置 5 套渐变配色方案，一键切换；Logo 背景色和标题文字颜色也可自定义。
- 🖼️ **Logo 管理**：支持使用 Clearbit 自动获取网站 Logo，也可上传本地图片或输入图片链接。
- 📱 **社交适配**：卡片比例为 3:4（900×1200px），完美适配小红书图文笔记。
- ⬇️ **一键导出**：基于 `html2canvas` 渲染为高清 PNG 图片，方便保存分享。

## 🚀 快速上手

1. **打开页面**  
   直接运行 `vibe-card-generator.html` 文件（推荐使用 Chrome / Edge 等现代浏览器）。

2. **获取网站信息**  
   - 在左侧输入框粘贴目标网站 URL（例如 `https://example.com`）。  
   - 点击 **“读取网站信息”** 按钮，程序会自动填充标题、描述、Logo 和域名。

3. **手动微调**  
   - 修改任意文本框（标题、tagline、详细介绍、标签等）即可实时预览卡片变化。  
   - 切换配色主题、调整 Logo 背景色或标题字体颜色。  
   - 支持上传本地 Logo 图片（覆盖自动获取的 Logo）。

4. **导出图片**  
   点击 **“导出 PNG”** 按钮，等待几秒即可下载卡片图片（文件名为工具名称）。

## 🧰 自定义说明

| 字段 | 说明 |
|------|------|
| 工具名称 | 卡片的标题，默认抓取网站 `<title>` 或 og:title |
| 一句话定位 | 显示在标题下方，常用于强调核心卖点 |
| 详细介绍 | 支持多行文本，描述工具功能与亮点 |
| 标签 | 逗号分隔，第一个标签会高亮显示 |
| 期数 / 日期 | 显示在卡片顶部和底部，便于系列化发布 |
| 网址显示文本 | 卡片底部浏览器栏显示的域名 |

> 所有修改都会实时同步到右侧卡片预览区。

## 🎨 配色主题

内置 5 套渐变配色（主色 + 辅色）：
- 🌿 青柠 + 珊瑚（默认）
- 💠 电光蓝 + 品红
- ☀️ 暖阳 + 玫瑰
- 🌸 紫雾 + 薄荷
- 🌲 森绿 + 橙

你可以通过点击色块快速切换，并单独调整 Logo 背景和标题颜色。

## 🛠️ 技术栈

- **HTML5 + CSS3**：卡片布局与交互动效
- **原生 JavaScript**：数据绑定、DOM 操作、图片抓取
- **[html2canvas](https://html2canvas.hertzen.com/)**：将 DOM 节点渲染为 Canvas 并导出 PNG
- **[Clearbit Logo API](https://clearbit.com/logo)**：自动获取网站 Logo（需网络）
- **[AllOrigins](https://allorigins.win/)**：用于绕过跨域限制抓取网页元数据
